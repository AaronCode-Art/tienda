#estructura del proyecto

components: Piezas reutilizables de la interfaz (botones, tarjetas de productos, barras de navegación).

pages: Componentes que representan una ruta completa (Inicio, Detalles del Producto, Carrito, Checkout).

hooks: Lógica personalizada de React (por ejemplo, useCart para manejar las compras).

services: Funciones para interactuar con tu base de datos o API.

types: Definiciones de interfaces de TypeScript (cómo luce un Producto, un Usuario, etc.).

context: Para el manejo del estado global, como el carrito de compras que debe verse en toda la web.

en este presente proyecto se vera reflejado el avance en equipo usando los commits y control de versiones con gitlab 

se uso react + typescript con reat compilator
